#  Getting started

## FRONT 

Start the front end Nextjs app

### 1 init

```
npm install
# or
yarn

```

### 2 run

```
npm run dev
# or
yarn dev

```

## BACKEND 

Start the back end StrapiJs app

### 0 Database

```
- Check the .env.local file to create a mysql database with the specified values

- Clone the .env.local file and rename it .env

```

### 1 init

```
npm install
# or
yarn

```

### 2 run

```
npm run develop
# or
yarn develop

```
